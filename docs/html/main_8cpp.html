Detailed Description

Game controller using Wi-Fi, MQTT, NeoPixel LEDs, and an RFID module.

This program connects to an MQTT server, communicates with various hardware components like NeoPixels and an RFID reader, and handles game logic where players input sequences of button presses.

The program also interfaces with OLED displays, updates via MQTT, and handles game sequences.
