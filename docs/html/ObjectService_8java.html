Detailed Description

The ObjectService class manages game-related logic, including operations involving players, controllers, sessions, and points.

The ObjectService class manages game-related logic, including operations involving players, controllers, sessions, and points. It interacts with the repository layer to perform CRUD operations, manage game sessions, update player scores, and handle game rounds. This class includes methods for creating new rounds, updating player points, fetching current rounds, and determining round winners.
